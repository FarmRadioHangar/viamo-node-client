create(params)

Add a new language.

Parameters:
Name Type Description
params Object

An object with query parameters to be sent with the Viamo API request. The query parameters available for this request are:

Name Required Type Description
name Yes String Name of the language.
abbreviation No String E.g., "EN" for English
Source:
  • modules/languages.js, line 52
See:
  • https://go.votomobile.org/apidoc/languages.html#create-languages
Returns:

A Promise that resolves to the response object from Viamo.

Example
// Add a new language

viamo.languages.create({
  name: 'Amharic',
  abbreviation: 'AM'
})
.then((res) => {
  if (200 == res.status) {
    console.log(res.message);
  }
  return viamo.languages.get();
})
.then((res) => {
  res.data.languages.forEach((language) => {
    console.log(language.name);
  });
});
